Included in v0.18.0
-------------------
1. Platform Owner Admin UI
   - Added dedicated Platform Owner tab/surface.
   - Kept platform-wide controls separate from tenant/organisation controls.

2. Platform Dashboard
   - Organisation counts.
   - Active/suspended tenant counts.
   - Plan count.
   - Recent platform audit events.
   - Recent platform security events.
   - API health/readiness summary.

3. All Organisations UI
   - List and search organisations.
   - Create new organisations.
   - Activate/suspend organisations.
   - View plan summary per organisation.
   - Assign plans to organisations.
   - Apply plan defaults to organisation features/limits.

4. Global Plans / Features UI
   - View global plan catalogue.
   - Create/update plans.
   - Edit feature defaults using JSON.
   - Edit plan limits using JSON.

5. Cross-tenant Logs
   - Platform audit log viewer.
   - Platform security event viewer.
   - Exact action/event filter support.

6. Platform System / Developer UI
   - API health and readiness links.
   - Swagger/OpenAPI links.
   - Current platform auth and permission context.

7. Frontend API Service Helpers
   - Added platform organisation helpers.
   - Added platform audit/security log helpers.
   - Added platform plan assignment/default helpers.
   - Added global plan create/update helpers.
