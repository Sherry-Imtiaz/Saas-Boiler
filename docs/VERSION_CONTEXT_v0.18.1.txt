Included in v0.18.1
-------------------
1. Windows Script Stabilisation
   - Quoted script working directory changes for paths containing spaces.
   - Updated install, seed, check and reset helper scripts.

2. Local Validation Command
   - Added validate-local.cmd.
   - Added root npm script: npm run validate:local.
   - Validates setup files, build, unit tests and deployment readiness in skip-db mode.

3. Integration Test Stabilisation
   - Added test-integration-local.cmd for Docker/local MongoDB testing.
   - Added test-integration-memory.cmd for optional mongodb-memory-server testing.
   - Updated integration test setup so local/Docker MongoDB is the default.
   - mongodb-memory-server is now opt-in via USE_MONGODB_MEMORY_SERVER=true.

4. Test Environment Documentation
   - Updated apps/api/.env.test.example.
   - Added docs/LOCAL_VALIDATION_RUNBOOK.md.

5. Version Metadata
   - Updated package versions to 0.18.1.
   - Updated API health/schema/deployment/check metadata to 0.18.1.
   - Added static OpenAPI JSON/YAML files for v0.18.1.

Validation
----------
- npm install --ignore-scripts --no-audit --no-fund
- npm run build
- npm test

Database-backed checks should be run on the user's Windows machine with Docker/MongoDB:
- docker compose up -d
- npm run seed
- npm run check:install
- npm run test:integration:local
